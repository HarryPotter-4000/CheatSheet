Циклы

«while» :

while (condition) {код также называемый "телом цикла"}

Код из тела цикла выполняется, пока условие condition истинно.

«do …while» :

do {тело цикла} while (condition)

Цикл сначала выполнит тело, а затем проверит условие (condition), // и пока его значение равно true, он будет выполняться снова и снова.

«for» :

for (начало; условие; шаг) {... тело цикла ...}

Цикл for повторяет действия, пока не произойдёт какое-либо специальное событие завершения цикла.

«for…in» :

for (key in object) {тело цикла выполняется для каждого свойства объекта}

Оператор for...in проходит по всем перечислимым свойствам объекта.

«for... of» :

for (variable of object) {выражения}

Оператор for...of создаёт цикл, проходящий по перечислимым объектам.
